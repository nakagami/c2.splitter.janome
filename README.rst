c2.splitter.janome
====================

Installation
--------------------

Add ``c2.splitter.janome`` to the list of eggs to install, e.g.::

   [buildout]
   ...
   find-links +=
       ...
       https://github.com/nakagami/c2.splitter.janome/tarball/master#egg=c2.splitter.janome

   ...
   eggs =
       ...
       c2.splitter.janome

Re-run buildout, e.g. with::

   $ ./bin/buildout
